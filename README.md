# "Ndeli"

## サイト概要
### サイトテーマ
レシピ掲載コミュニティサイト

### テーマを選んだ理由
病気などで食事制限が必要で日々の食事などに困っている方や、ダイエット、体質改善をしたい方に向けて自信の体に合わせたレシピを共有できるサービスを考えました。
便利に自身に合わせたレシピを選択できることにより日々の食生活の向上につながると考えます。

### ターゲットユーザ
- 病気などで食事制限が必要な方
- 体質改善が必要な方
- 自身のレシピを共有したい方

### 主な利用シーン
- 料理をするとき
- 自身のレシピを掲載する時
